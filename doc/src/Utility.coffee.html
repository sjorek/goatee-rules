Static method parseRules

NON-STANDARD caching lightweight version of CSSOM.CSSStyleRule.parse

Parameters:

  • code must be a String.

Returns an Expressions

Utility.parseRules = do -> parser = null cache = {} (code) -> return cache[code] if cache.hasOwnProperty(code) parser ?= require './Rules' rules = parser.parse code cache[code] = cache['' + rules] = rules
